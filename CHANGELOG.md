# 0.0.2

- 修正`反向电阻计算`部分整数电阻值无法计算出
- 添加超时逻辑，避免部分原理图PCB操作卡死

# 0.0.1

- 初始发布
- 主要功能：
    - PCB 线宽与过孔尺寸计算
    - 在 PCB 界面读取并应用线宽与过孔（尚未实现）
    - 分压计算：根据电压计算分压电阻；根据电阻计算分压电压
    - 一键在原理图放置元件（受 EDA bug 限制，部分功能暂未实现）
    - 非隔离开关电源电感值计算，并提供商城搜索入口
    - 原理图中一键放置常用元件：
        - Type-C 及 CC 下拉电阻
        - 彩色 LED 与限流电阻
        - I2C 上拉电阻
        - USB 接口（贴片插件 / 公头 / 母座 / USB2 / USB3）
    - PCB 差分对快速创建与疑似差分对识别
